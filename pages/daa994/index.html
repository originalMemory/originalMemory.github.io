不同的二叉搜索树

# 描述

96. 不同的二叉搜索树 (opens new window)

给你一个整数 n ，求恰由 n 个节点组成且节点值从 1 到 n 互不相同的 二叉搜索树 有多少种？返回满足题意的二叉搜索树的种数。

示例 1：

输入：n = 3
输出：5
1
2

示例 2：

输入：n = 1
输出：1
1
2

提示：

  • 1 <= n <= 19

# 解法

# 动态规划

设序列 1⋯n1 \cdots n1⋯n ，生成数时，可以通过遍历数字 iii ，以该数为根， 1⋯(i−1)1 \cdots (i - 1)1⋯(i−1) 为左子树， (i+1)⋯n(i + 1) \cdots n(i+1)⋯n 为右子树，其可能生成的树数量的值累加即为总的数量。且这两棵子数的数量即为对应序列的值的，设 G(n)G(n)G(n) 是树的总数，是 F(i)F(i)F(i) 为 iii 为根的树数量，由此可得。

G(n)=∑i=1nF(i)G(n) = \sum_{i=1}^nF(i) G(n)=i=1∑n​F(i)

F(i)=G(i−1)∗G(n−i)F(i) = G(i-1)*G(n-i) F(i)=G(i−1)∗G(n−i)

G(0)=1,G(1)=1G(0)=1,G(1)=1 G(0)=1,G(1)=1

代码：

class Solution:
    def numTrees(self, n):
        G = [0] * (n + 1)
        G[0], G[1] = 1, 1

        for i in range(2, n + 1):
            for j in range(1, i + 1):
                G[i] += G[j - 1] * G[i - j]

        return G[n]

# 卡塔兰数 (opens new window)

上述算法在数学上被称为卡塔兰数 CnC_nCn​ ，其公式为：

C0=1,Cn+1=2(2n+1)n+2CnC_0=1, C_{n+1}=\frac{2(2n+1)}{n+2}C_n C0​=1,Cn+1​=n+22(2n+1)​Cn​

代码：

class Solution:
    def numTrees(self, n):
        for i in range(0, n):
            C = C * 2 * (2 * i + 1) / (i + 2)
        return int(C)
